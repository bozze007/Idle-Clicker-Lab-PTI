Aturan Gameplay:
game ini menggunakan konsep clicker, ketika pemain meng-klik gambar the rock maka akan mendapatkan uang. uang yang terkumpul sampai sekian, maka pemain dapat membeli upgrade. 
ketika uang sudah sampai 100, maka pemain dapat membeli auto clicker. setelah membeli upgrade, pendapatan dalam satu kali klik juga akan bertambah. Pemain akan mendapatkan 
peringatan jika ingin membeli upgrade dan/atau autoclicker ketika uang yang dimiliki tidak mencukupi. selanjutnya pada halaman about us, akan memperlihatkan data kelompok,
yaitu kelompok KAMIS PAGI.
